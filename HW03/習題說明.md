nn0.py 的原理
這份 nn0.py 是一個輕量級的深度學習框架實作，核心圍繞著**自動微分（Autograd）**與 Transformer 類神經網路組件。其設計理念與 PyTorch 的 torch.tensor 或 Micrograd 類似，透過建立運算圖來實現反向傳播。

以下是該程式碼的四大核心原理詳解：

1. 自動微分引擎：Value 類別
這是整個框架的靈魂，利用**運算子過載（Operator Overloading）**來記錄數學運算的軌跡。

動態運算圖建構：
每個 Value 物件不僅儲存數值（data），還儲存了它的來源（_children）以及對應的局部導數（_local_grads）。
當您執行 a + b 或 a * b 時，系統會自動建立一個新的 Value 節點，將 a 與 b 連接起來。
拓撲排序（Topological Sort）：
在 backward() 函式中，系統先透過 build_topo 演算法將所有節點排序。這確保了在計算梯度時，父節點的梯度一定會在子節點之前計算完成。
鏈式法則（Chain Rule）實作：
梯度計算的核心邏輯為：child.grad += local_grad * v.grad。這代表當前的全域梯度是「局部變化率」與「後續節點傳回來的梯度」之乘積。
2. 優化演算法：Adam 優化器
Adam (Adaptive Moment Estimation) 是目前深度學習最常用的優化器，nn0.py 完整實作了其動態調整學習率的機制。

一階與二階動量：
m (Momentum)：記錄梯度的指數移動平均，幫助跨越局部最小值。
v (Velocity)：記錄梯度平方的移動平均，用來調整每個參數的縮放比例。
偏差修正（Bias Correction）：
透過 m_hat 與 v_hat 的計算，修正訓練初期動量值偏向 0 的問題。
學習率衰減：
在 gd() 函式中，實作了線性衰減邏輯：lr_t = optimizer.lr * (1 - step / num_steps)，這有助於模型在訓練後期更穩定地收斂。
3. Transformer 關鍵組件
程式碼中包含了一些專為類 Transformer 模型設計的現代化算子：

rmsnorm (Root Mean Square Layer Normalization)：
與傳統 LayerNorm 不同，它只進行縮放而不進行平移（Offset），計算效率更高，這也是 Llama 等現代大模型偏好的校準方式。
數值穩定性 softmax：
實作中先減去 max_val 再進行 exp() 運算。這是為了防止指數運算導致數值溢位（Floating point overflow），確保計算機在處理大數值時不會崩潰。

4. 訓練流程：gd() 函式
這是模型的訓練循環（Training Loop）核心，體現了「隨機梯度下降」的現代變體：

Forward Pass：將 Token ID 轉為預測機率（Logits）。
Loss Calculation：使用負對數似然（Negative Log-Likelihood）作為損失函數：loss_t = -probs[target_id].log()。
Backward Pass：呼叫 loss.backward() 觸發整條鏈的自動微分。
Update：透過 optimizer.step() 應用 Adam 更新參數並清空梯度，準備下一次迭代。
